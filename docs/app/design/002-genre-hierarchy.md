# 設計ドキュメント 002: ジャンルの2階層化

**Issue**: #3 ジャンルを細分化
**作成日**: 2026-02-28
**ステータス**: レビュー待ち

---

## 概要

現在の11ジャンル1階層構造を、大ジャンル＋小ジャンルの2階層構造に拡張する。
書籍数が多い大ジャンル（フィクション: 160冊、ノンフィクション: 157冊など）をサブジャンルでさらに絞り込めるようにする。

### 現状の問題

| ジャンル | 冊数 | 問題 |
|---------|------|------|
| フィクション | 160 | 絞り込み効果が薄い |
| ノンフィクション | 157 | 絞り込み効果が薄い |
| フィクション（日本） | 147 | 絞り込み効果が薄い |
| 漫画・コミック | 116 | 絞り込み効果が薄い |
| SF | 78 | 現状で十分絞り込める |
| 未分類 | 64 | サブジャンルの整備で削減可能 |
| エッセイ | 43 | 現状で十分 |
| 実用 | 35 | 現状で十分 |

## 技術的アプローチ

### フォルダ構造の現状

現在の Google Drive フォルダ構造は以下のとおり。サブジャンルを示すサブフォルダは存在しない。

```
70_Book / 72_BookscanPdf / 72_BookscanPdf files / フィクション
70_Book / 72_BookscanPdf / 72_BookscanPdf files / ノンフィクション
70_Book / kindle / フィクション
70_Book / kindle / フィクション / 日本の作家   ← 唯一のサブフォルダ例
```

フォルダパスのみからサブジャンルを判定するには **Google Drive 側のフォルダ再編** が必要になる（819冊分の手作業）。

### アプローチ比較

| | アプローチ A: フォルダ構造拡張 | アプローチ B: タイトル・著者キーワード推定 |
|--|------|------|
| **概要** | Google Drive にサブジャンルフォルダを作り直す | タイトル・著者名・シリーズ名のキーワードマッチでサブジャンルを推定する |
| **精度** | 高（フォルダ分け時に人手で確認） | 中（ルール設計次第） |
| **初期コスト** | 高（全819冊のフォルダ移動） | 低（ルール定義のみ） |
| **維持コスト** | 低（追加登録時にフォルダを指定） | 中（新規書籍でルールが足りない場合に調整） |
| **自動化** | process.js の変更が最小 | process.js にキーワードルールを追加 |
| **推奨** | — | **推奨** |

### 推奨: アプローチ B — キーワードベース自動推定

## 変更範囲

### 1. `data/books.json` スキーマ変更

`subgenre` フィールドを追加する。

```json
// 変更前
{
  "genre": "フィクション"
}

// 変更後
{
  "genre": "フィクション",
  "subgenre": "ミステリー"
}
```

`subgenre` が判定できない場合は `null` とする。

#### スキーマ設計の比較

| | 案1: 独立フィールド（推奨） | 案2: ネストオブジェクト |
|--|------|------|
| **構造** | `"genre": "...", "subgenre": "..."` | `"genre": { "main": "...", "sub": "..." }` |
| **後方互換性** | 既存の `genre` フィールドをそのまま維持 | 既存コードの破壊的変更が必要 |
| **UI実装の容易さ** | シンプル | やや複雑 |
| **推奨** | **推奨** | — |

### 2. `scripts/process.js` 変更

F-2（ジャンル推定）を拡張し、`subgenre` を推定するロジックを追加する。

```
現在: F-2 → genre のみを割り当て
変更後: F-2 → genre を割り当て → F-2b: subgenre を割り当て
```

### 3. UI 変更（フィルタ機能）

書籍一覧（S-1）のジャンルフィルタを2階層対応にする。

```
【現在】
  ジャンル: [全て] [フィクション] [ノンフィクション] ...

【変更後（案）】
  大ジャンル: [全て] [フィクション] [ノンフィクション] ...
      ↓ フィクションを選択すると
  小ジャンル: [全て] [ミステリー] [歴史・時代] [その他フィクション]
```
